## 项目规范

### 增加文档规范

现在文档书写流程简化成只需要配置根目录下的 `sidebarConfig.js` 文件，内附说明。

> 注意，每个文档目录需要添加一个 `index.mdx` 入口文件，对应落地页，可添加一段描述，例：`design-pattern/index.mdx`
