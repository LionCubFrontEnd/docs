# 小狮子前端编程成长圈

## 站点介绍

哈喽，大家好，这里是 Chocolate 和 HearLing 一起搭建的前端编程成长圈，我们希望通过这个站点汇聚一些好学的编程爱好者，扩大前端圈。

在这里我们要介绍的是我们整站是不会添加弹窗广告的，**美观与简洁**是我们自己也比较注重的。

同时，作为前端程序员的我们，也有开源精神，我们的初衷就是提供一份**完整、详细**的学习笔记文档，并且是**免费**的，对于大多数白嫖党来说，你完全可以按照网站分类进行学习，我们也希望我们的文档能够帮助到大家找到一份好的工作，亦或是技能上与认知上的提升。

> 当然，可能会有人疑问了，真就做一个完全没有收益的网站吗？

这个倒不会，我们创建了属于我们自己的[知识星球](https://docs.yangchaoyi.vip/zsxq)，也希望能够收获一批铁杆粉丝。在这里面，我们会分享我们过往的面试求职经经验，读书心得，问答。与此同时，我们都是 B 站的 UP 主，今后应该都会一起联动多多更新视频，也会有直播做项目，在星球里面会提供完整的直播回放，手把手的做一些企业级项目，一起学习前端。


### 初心

工作之后发现生活的节奏快了很多，尤其是在一线城市里，每天工作完之后就很难沉下心来去学习与提升，我们自己也常常反思为什么明明道理都懂，目标也定下来了，却一直没有落实到位。

因此，我们采用了自己搭建网站，一起共建社区的形式驱动自身去保持对于生活与工作的平衡。

我们也清楚自身也许并没有那么优秀，比我们技术牛的，懂的多的实在太多太多了，优秀的人有许多。但我们**依旧觉得我们建立社区不是说我们有了怎样的技术或者到达怎样的认知才行，而是在这个过程中不断地提升，让我们自己变的越来越优秀**，于是，我们想明白了，我们就想要建立这样的社区，希望与社区一起成长。

毕竟会有付费用户，所以我们也会更加用心对待我们的用户，也愿意和大家交个朋友，提供一些帮助与咨询。

我们的文档网站是完全免费的，我们也会把内容做的更好，一个关键点就是【遇见】，希望来访的各位小伙伴在这里能够通过一站式的学习，能够有技术提升，找到一份不错的工作那就更好啦。



### 求合作

整理网站的内容不易，大家也可以看到，我们将内容进行了明确的划分，可以根据目前所学的知识**选择性阅读**。

这份笔记是我们今后要做持续维护的，我们会不断的完善它，让它更受大家的喜欢。因此，我们也愿意和大家一起合作把内容做的更好。

尽管我们编写笔记、博客内容多年，但可能还是会有笔误或者内容上的不足，欢迎你通过留言的方式给我们反馈问题。

同样，你也可以提供一些需求，包括提供一些网站建设的想法，新功能等，我们很愿意听取各位的建议。

### 优先级

我们优先级是先将免费的文档网站给做好，这也是必须的。

网站功能也是优先我们自己的想法，其次再是大家提供的建议与改进（可能有的话）。
